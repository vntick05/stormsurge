# Storm Surge Studio V4

Standalone V4 UI workspace for Storm Surge.

This app was bootstrapped from the MIT-licensed Mantis free React admin template:
https://github.com/codedthemes/mantis-free-react-admin-template

The existing Storm Surge UIs were left untouched. This directory is a separate Vite/React app intended as a clean starting point for V4 work.

## Run

```bash
yarn install
yarn dev
```

Default local URL: `http://localhost:3004`

## Notes

- The upstream nested `.git` history was not copied into this repo.
- The app base path was reset to `/` instead of the vendor demo path.
- Browser auto-open was disabled for local dev and preview.
